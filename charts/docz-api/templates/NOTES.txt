docz-api {{ .Chart.AppVersion }} has been deployed as release "{{ .Release.Name }}".

The container serves everything on one HTTP port: the API, health probes,
the OpenAPI spec, and Prometheus metrics.

1. Port-forward the service to reach it locally:

     kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "docz-api.fullname" . }} 8080:{{ .Values.service.port }}

2. Then browse:

     API:          http://localhost:8080/api/v1
     Liveness:     http://localhost:8080/healthz
     Readiness:    http://localhost:8080/readyz
     OpenAPI spec: http://localhost:8080/openapi.yaml
{{- if .Values.metrics.enabled }}
     Metrics:      http://localhost:8080/metrics
{{- end }}
